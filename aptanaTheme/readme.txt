1. Download bundle.rb to /home/USER/Documents/Aptana Rubles/theme.ruble/bundle.rb (WIN User: C:\Benutzer\diego\Aptana Rubles\theme.ruble\my-bundle.rb)

4. Restart Aptana and open a php file ... the php color syntax highlighting sould now be better :)
5. Have fun
